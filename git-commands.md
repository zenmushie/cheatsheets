
```

  git branch -av #shows all in verbose mode
```

``` 
  git branch {{branch_name}} #create a new branch
```

```
  git branch -d {{branch_name}} #delete a branch, error if not merged
```
```
  git branch -D {{branch_name}} #force delete a branch
```
